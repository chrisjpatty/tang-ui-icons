# tang-ui-icons

> Icons for the tang-ui library by @chrisjpatty

[![NPM](https://img.shields.io/npm/v/tang-ui-icons.svg)](https://www.npmjs.com/package/tang-ui-icons) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save tang-ui-icons
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'tang-ui-icons'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
